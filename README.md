# Анализ спроса и факторов, влияющих на статус бестселлера на Amazon
#### Описание проекта
Проект направлен на анализ факторов, связанных с уровнем спроса и статусом бестселлера товаров на маркетплейсе Amazon на основе открытых данных (1,4 млн товаров за 2023 год).

#### Задачи
* Проанализированы распределения цен и спроса на товары маркетплейса.
* Определены ключевые характеристики товаров, связанные со статусом бестселлера.
* Проведено сравнительное исследование параметров бестселлеров и небестселлеров с применением статистических тестов.
* Оценено влияние скидок, рейтинга и количества отзывов на вероятность попадания товара в категорию Бестселлеров.
* Построена интерпретируемая модель бинарной классификации для количественной оценки выявленных зависимостей.

#### Стек проекта
Python (Pandas, Numpy, SciPy, Seaborn, Matplotlib, Scikit-learn)

#### Проект включает
* Загрузку и предобработку данных - загрузка через Kaggle API, проверка пропусков и дублей, объединение таблиц товаров и категорий;
* Проведение разведочного анализа данных (EDA) - анализ распределений цены, спроса, отзывов и рейтинга, а также логарифмирование и capping экстремальных значений;
* Анализ различий между бестселлерами и небестселлерами - тест Манна–Уитни для проверки значимости различий между группами, анализ топ-10 категорий по доле бестселлеров;
* Разработку интерпретируемой модели бинарной классификации - логистическая регрессия с балансировкой классов, оценка по ROC-AUC и PR-AUC, интерпретация коэффициентов через odds ratio;
* Выводы и рекомендации.

#### Основные результаты
* Рейтинг товара - наиболее значимый фактор: медиана у бестселлеров 4.6 против 4.4 у обычных товаров, увеличение на одно стандартное отклонение повышает шансы стать бестселлером в 2.7 раза
* Скидка положительно связана со статусом бестселлера: медиана скидки у бестселлеров 24% против 17% у обычных товаров
* Цена оказывает отрицательное влияние: более высокая цена снижает вероятность попадания в бестселлеры
* Наблюдается выраженная неоднородность доли бестселлеров между категориями
* ROC-AUC модели составил 0.689; PR-AUC = 0.014 при базовом уровне 0.006, что отражает реальную сложность задачи при сильном дисбалансе классов (доля бестселлеров < 1%)

#### Ограничения модели
Модель отражает статические зависимости на момент наблюдения и не учитывает динамику продаж, маркетинговые активности, сезонность и другие внешние факторы. Для практического применения требуется расширение признаков и дополнительная валидация.
