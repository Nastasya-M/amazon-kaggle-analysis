# Анализ спроса и факторов, влияющих на статус бестселлера на Amazon
#### Описание проекта
Определить ключевые факторы, влияющие на попадание товара в категорию "Бестселлер" и построить интерпретируемую модель, позволяющую оценивать вероятность того, что товар станет бестселлером на основе определенных характеристик.

#### Задачи:
* Проанализированы распределения цен и спроса на товары маркетплейса.
* Определены ключевые характеристики товаров, связанные со статусом Бестселлера.
* Проведено сравнение ключевых параметров бестселлеров и небестселлеров.
* Оценено влияние скидок, рейтинга и количества отзывов на вероятность попадания товара в категорию Бестселлеров.
* Построена модель, позволяющая анализировать связь параметров товара со статусом Бестселлера.

#### Стек проекта
Python (Pandas, Numpy, Seaborn, Matplotlib, Sklearn)

#### Проект включает:
* Загрузку и первичную обработку данных маркетплейса через Kaggle API;
* Проведение разведочного анализа данных (EDA);
* Анализ различий между бестселлерами и небестселлерами;
* Разработку интерпретируемой модели бинарной классификации;
* Выводы и рекомендации.
